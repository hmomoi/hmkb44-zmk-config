# hmkb44 のページ

44キーのキーボード「hmkb44」の作成資材のページです。

主に、zmkファームウェアの管理と、KiCadの設計資材をおいています。

# 目指すキーボード
* 40%キーボード（EPOMAKER TH40を参考に）
* Bluetoothで接続
* LEDチカチカ
* charlieplexの実験

このくらいを目指して作ってます。
seeeduino_xiao_ble を使ってイロイロ実験してみたくて作成しました。
